Les objectifs du programme

Pogramme de remplissage d'une salle de cinéma, qui permet de réserver des places et d'afficher la salle au fur et à mesure des réservations. La description détaillée de l'attendu est disponible dans le brief sur le site Simplonline.

La conception

Le programme est réalisé en java. Il utilise des méthodes basiques facilement compréhensibles pour un débutant.

Le code réalisé

Le programme est en cours de développement. Nous présentons ci-dessous les fonctionnalités déjà existantes.

Paramétrage de la salle

L'objectif de ce code est de rendre les dimensions de la salle facilement paramétrables:

// Dimension (paramétrables)
int totalNbRow = 20;
int totalNbSeatByRow = 25;

Choix de la rangée

La personne qui réserve peut choisir la rangée

// Choice of row
int chosenRow = -1;
while (chosenRow < 1 || chosenRow > totalNbSeatByRow) {
System.out.println("\nChoississez votre rangée (1 à" + totalNbRow +"):");
chosenRow = sc.nextInt();
sc.nextLine();
if (chosenRow < 0 || chosenRow > totalNbRow) {
System.err.println("\nAttention, cette rangée n'existe pas!\n");
}
}

Choix du nombre de sièges

La personne qui réserve peut choisir le nombre de siège

// Choice of nb of seat
int seatNumber = -1;
while (seatNumber < 1 || seatNumber > 7) {
System.out.println("Nombre de sièges (1 à 10)");
seatNumber = sc.nextInt();
sc.nextLine();
if (seatNumber < 1 || seatNumber > 7) {
System.err.println("Attention, le nombre maximum de siège est 10");
}
}

Fonction d'affichage de la salle

La personne qui réserve peut voir la salle avant validation

public static void roomOverview (String[][] movieRoom) {
for (int row = 0; row < 5; row++) {
for (int seat = 0; seat < 7; seat++) {
System.out.printf("[%3s]", movieRoom[row][seat]);
}
System.out.println();
}
}
